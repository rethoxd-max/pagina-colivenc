RESUMEN

Este Trabajo de Fin de Grado presenta el desarrollo de una aplicación web completa para la gestión deportiva, implementada utilizando tecnologías modernas como Angular para el frontend y MongoDB como base de datos, con un backend desarrollado en Node.js. El sistema permite la gestión integral de atletas, competiciones, rankings y entrenamientos, proporcionando una solución completa para clubes deportivos y federaciones. La aplicación facilita el seguimiento del rendimiento de los atletas, la organización de competiciones y la gestión de entrenamientos, todo ello a través de una interfaz intuitiva y responsive.

El proyecto implementa un sistema de autenticación robusto con diferentes roles de usuario (Administrador, Entrenador, Atleta, Editor y Viewer), garantizando la seguridad y el acceso controlado a las diferentes funcionalidades. Además, incorpora características avanzadas como la gestión de rankings por categorías, un sistema de inscripciones a competiciones y un calendario de entrenamientos personalizado.

Palabras clave: Angular, MongoDB, Node.js, Gestión Deportiva, Desarrollo Web, Express, REST API

ABSTRACT

This Final Degree Project presents the development of a comprehensive web application for sports management, implemented using modern technologies such as Angular for the frontend and MongoDB as a database, with a backend developed in Node.js. The system allows comprehensive management of athletes, competitions, rankings and training, providing a complete solution for sports clubs and federations. The application facilitates athlete performance tracking, competition organization and training management, all through an intuitive and responsive interface.

The project implements a robust authentication system with different user roles (Administrator, Coach, Athlete, Editor and Viewer), ensuring security and controlled access to different functionalities. Additionally, it incorporates advanced features such as category-based ranking management, a competition registration system and a personalized training calendar.

Keywords: Angular, MongoDB, Node.js, Sports Management, Web Development, Express, REST API

7. SEGURIDAD

7.1. Autenticación y autorización

Implementación de seguridad robusta:

```javascript
// Middleware de autenticación
const authMiddleware = async (req, res, next) => {
  try {
    const token = req.header('Authorization').replace('Bearer ', '');
    const decoded = jwt.verify(token, process.env.JWT_SECRET);
    const user = await User.findOne({ _id: decoded._id });
    if (!user) throw new Error();
    req.user = user;
    next();
  } catch (error) {
    res.status(401).send({ error: 'Please authenticate.' });
  }
};

// Control de acceso basado en roles
const roleMiddleware = (allowedRoles) => {
  return (req, res, next) => {
    if (!req.user.userTypes.some(role => allowedRoles.includes(role))) {
      return res.status(403).json({ message: 'Access denied' });
    }
    next();
  };
};
```
